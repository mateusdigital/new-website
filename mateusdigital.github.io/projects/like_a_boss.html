Like a Boss

It is the studio's main game, Like a Boss is an Action RPG where you are the epic Raid Boss at the end of a dungeon, crushing players (the heroes) as they come to steal your treasures.

As an additional programmer, I was responsible for numerous code refactoring, improvements, bug fixes, and other optimizations that are typically not available to the main team.

I created custom Editor Scripts to ease some game development operations.
